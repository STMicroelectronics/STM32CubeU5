Purpose

These drivers provide a set of functions to manage:

  • LEDs, push-buttons, joystick, potentiometer and COM ports
  • Camera module
  • Audio IN and OUT
  • IO Expander
  • LCD module with Touchscreen
  • SD card
  • OctoSPI NOR Flash memory
  • SRAM memory
  • Motion sensors
  • EEPROM memory
  • USBPD Type C port

Note that stm32u575i_eval_conf_template.h file must be copied in user application as stm32u575i_eval_conf.h with optional configuration update.
