Purpose

These drivers provide a set of functions to manage:

  • LEDs, push-button, and COM ports
  • OctoSPI NOR Flash memory
  • HexaSPI PSRAM memory
  • LCD
  • Touch Screen
  • eMMC memory
  • Ranging sensor
  • Environmental sensor

Note that stm32u5x9j_discovery_conf_template.h file must be copied in user application as stm32u5x9j_discovery_conf.h with optional configuration update.

Update History

Main Changes

  • Add stm32u5x9j_discovery_env_sensors driver to support Temperature sensor
  • Update stm32u5x9j_discovery_hspi.c bsp driver by HSPI tuning value
  • MISRA-C 2012 and MCUAstyle coding rules compliance Formatting
  • STM32U5x9J-DK_BSP_User_Manual.chm: Updated

Dependencies

This software release is compatible with:

  • BSP common drivers V7.2.1 or above

Limitations

  • Multitcouch functionality not supported
  • eMMC driver will be subject to optimization in the next release
