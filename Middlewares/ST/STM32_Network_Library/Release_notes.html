Purpose

The STM32 Network Library is a middleware providing network services on STM32 devices. It provides a socket API (BSD like style) with support of secure or non secure connection and an API to control the life cycle of the network adapters.

Two classes of network adapters are supported WIFI and Ethernet. Different WIFI modules are supported from third party vendors.
